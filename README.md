Hybrid AI-based Biotechnology Q&amp;A Chatbot using Python, SQLite, MeSH, and LLaMA (Groq API)
Biotechnology Q&A Chatbot

This is a hybrid AI-based chatbot that answers biotechnology questions using:

- SQLite database (for predefined answers)
- MeSH (NCBI biomedical terminology)
- LLaMA 3.1 (Groq API) for AI-generated responses

Features
- Answers questions like PCR, ELISA, CRISPR, DNA, etc.
- Hybrid architecture (Database + MeSH + AI)
- Chat-style interface using Streamlit
- Real-time responses

Tech Stack
- Python
- Streamlit
- SQLite
- Groq API (LLaMA 3.1)
- Biopython (MeSH)

Deployment
Deployed using Render for public access.
